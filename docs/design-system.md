
# Design System — Autonova

## Objetivo

Este documento define toda a identidade visual da plataforma Autonova.

Nenhum componente deve ser criado sem seguir estas regras.

O objetivo é manter consistência visual entre:

- Site institucional
- App
- Dashboard
- Futuro aplicativo mobile
- Painéis administrativos
- Landing Pages

---

# Filosofia

A Autonova deve transmitir:

- tecnologia
- inteligência
- confiança
- organização
- automação
- alto valor

A interface deve parecer premium.

Nunca deve parecer uma aplicação "genérica de administração".

---

# Princípios

## Minimalismo

Poucos elementos.

Muito espaço.

Muito respiro.

Nada poluído.

---

## Clareza

Toda informação importante deve ser imediatamente identificável.

---

## Hierarquia

Sempre existirão apenas três níveis:

Título

↓

Conteúdo

↓

Ações

Nunca misturar prioridades.

---

## Movimento

As animações existem para orientar.

Nunca para chamar atenção.
